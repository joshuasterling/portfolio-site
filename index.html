Unreal Skateshop

Unreal Skateshop is a fun e-commerce website that I created as my first personal project. The site is built with React utilizing Redux. It allows users to login to the site and add different parts to their cart to create a complete skateboard. Amazon S3 was utilized for all photos on the site and I implemented Stripe as a checkout method along with Nodemailer to notify the client of their purchase. The site follows RESTful standards and is complete with full CRUD.

Smash JAM

Smash JAM is a companion website to the popular video game "Super Smash Bros." The website is designed to allows users to login and see an updated News Feed, that dynamically displays article previews about the game from other popular websites. Users can explore all 82 characters in the game and view their stats, a short video about the fighter, counter-pick match ups and see all 8 of each characters unique costumes. All of this data is stored in a RESTful PostgreSQL database.

Users can also create and manage their own tournament on our site. The tournament creator allows users to select the size of the bracket from a dropdown of 4, 8, 16, 32 or 64 participants. Users can add/invite their friends to the event and all participants can join a Live Chat, utilizing Sockets. The entire site is fully responsive to mobile views.

*This site is not yet hosted as we're putting some final touches on it before release.
